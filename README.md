# FaceAttack
Black box attack to face recognition via M-DI2-FGSM  
本项目是一种人脸识别的黑盒攻击方法,项目正在完善中...

具体的攻击方法见[run2.py](https://github.com/zeng9820/FaceAttack/blob/main/run2.py)
